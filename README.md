# 基于Radxa ZERO 3W（RK3566）的 WIFI/4G 遥控摄像头小车（咪咪监控版）

### 基本功能

* Web远程操控
* 实时视频监控
* 录制和拍照
* 激光灯控制
* 定时和手动休眠
* 摄像头红外夜视自动切换
* 磁吸或者无线充电
* 24h+待机（非休眠）

### 基本原理

* 双电机控制，协同实现转弯与直行，开环控制
* Python Flask后端，Web前端
* Websocket可靠小车操控，ttp请求控制其他操作
* 后端ffmpeg硬件编码，通过HLS实时推流到前端播放

### 材料准备

1. 所有两驱或四驱直流有刷电机控制的小车
2. 瑞芯微（Rockchip）所有支持硬件视频编码（rkmpp）和WiFi的开发板，如RK3566/RK3568/RK3588
3. TB6612FNG电机驱动板
4. 树莓派500万像素红外夜视摄像头 IRCUT可调焦相机（OV5647）或其他与开发板兼容的摄像头模组（IRCUT夜视功能，可选）
5. INA226电压电流模块（R010，0.01Ω，MAX=8A）（测量电压电流用，可选）
6. DC-DC直流大功率12V转5V稳压电源
7. 12V 3S 锂电池
8. 激光灯模块（3PIN逻辑控制）
9. 12.6V无线充电模块或者磁吸充电接口（长期无人，遥控充电选用，可选）
10. USB 4G Dongle（4G CAT4，EC20，无wifi环境使用，可选)
11. Tailscale（异地组网，用于非局域网控制，可选）
12. I2S话筒和扬声器（用于语音对话，代码未实现，可选）
13. 其他物料包括但不限于：洞洞板，杜邦线，18/20AWG粗导线，M3内六角螺丝，热熔胶枪，铜柱，USB转TTL模块，5525 DC公母线，电烙铁、焊锡丝、助焊剂，万用表，读卡器等
